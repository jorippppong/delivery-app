## 일본어 단어 암기 노트 (이미지 기반)

아래 표는 첨부 이미지의 단어들을 정리한 것입니다.

| 단어         | 한자 구성                | 주요 부수             | 부수로 이미지 연상      | 뜻            | 발음         |
| ------------ | ------------------------ | --------------------- | ----------------------- | ------------- | ------------ |
| 音楽         | 音(소리) + 楽(즐거움)    | 音(소리), 木(나무)    | 소리와 악기(나무)       | 음악          | おんがく     |
| 交番         | 交(교차) + 番(차례)      | 亠(뚜껑), 田(밭)      | 길이 교차하는 순찰소    | 교번/순찰소   | こうばん     |
| 右           | 右(오른쪽)               | 口(입)                | 입 옆의 오른손          | 오른쪽        | みぎ         |
| 左           | 左(왼쪽)                 | 工(공구)              | 공구를 잡은 왼손        | 왼쪽          | ひだり       |
| 去年         | 去(떠나다) + 年(해)      | 厶(사사), 干(방패)    | 지난 해                 | 작년          | きょねん     |
| 答える       | 答(대답) + える          | 竹(대나무)            | 대로 만든 바구니로 답함 | 대답하다      | こたえる     |
| 宿題         | 宿(숙박) + 題(문제)      | 宀(지붕), 頁(머리)    | 집에서 하는 문제        | 숙제          | しゅくだい   |
| 夏           | 夏(여름)                 | 夊(천천히 감)         | 더워 천천히 걷는 여름   | 여름          | なつ         |
| 入口         | 入(들다) + 口(입)        | 入(들다), 口(입)      | 들어가는 입구           | 입구          | いりぐち     |
| 出口         | 出(나가다) + 口(입)      | 凵(그릇), 口(입)      | 나가는 출구             | 출구          | でぐち       |
| 大人         | 大(크다) + 人(사람)      | 大(큰 사람), 人(사람) | 큰 사람                 | 어른          | おとな       |
| 番号         | 番(차례) + 号(번호)      | 田(밭), 口(입)        | 차례를 매기는 번호      | 번호          | ばんごう     |
| 食べ物       | 食(먹다) + べ + 物(물건) | 食(먹다), 牛(소)      | 먹는 물건               | 음식          | たべもの     |
| 悪い         | 悪(나쁘다) + い          | 心(마음)              | 마음이 언짢음           | 나쁘다        | わるい       |
| 財布         | 財(재물) + 布(천)        | 貝(조개), 巾(천)      | 돈을 담는 천 주머니     | 지갑          | さいふ       |
| 話す         | 話(말) + す              | 言(말)                | 말로 풀어냄             | 말하다        | はなす       |
| 外国         | 外(밖) + 国(나라)        | 夕(저녁), 囗(에워쌀)  | 밖의 나라               | 외국          | がいこく     |
| 時計         | 時(시간) + 計(재다)      | 日(해), 言(말)        | 시간을 재는 기계        | 시계          | とけい       |
| ポケット     | -                        | -                     | 외래어                  | 주머니        | ポケット     |
| 寝る         | 寝(자다) + る            | 宀(지붕)              | 지붕 아래 누움          | 자다          | ねる         |
| 学校         | 学(배우다) + 校(학교)    | 子(아이), 木(나무)    | 아이가 배우는 곳        | 학교          | がっこう     |
| 電話         | 電(전기) + 話(말)        | 雨(비), 言(말)        | 전기로 하는 말          | 전화          | でんわ       |
| 手紙         | 手(손) + 紙(종이)        | 手(손), 糸(실)        | 손으로 쓴 종이          | 편지          | てがみ       |
| 午後         | 午(정오) + 後(뒤)        | 十(십), 彳(걷다)      | 정오 이후               | 오후          | ごご         |
| 今週         | 今(지금) + 週(주)        | 人(사람), 辶(길)      | 지금 이 주              | 이번 주       | こんしゅう   |
| エレベーター | -                        | -                     | 외래어                  | 엘리베이터    | エレベーター |
| 降りる       | 降(내리다) + りる        | 阝(언덕)              | 아래로 내려옴           | 내리다        | おりる       |
| 広い         | 広(넓다) + い            | 广(집 엄)             | 집이 넓게 트임          | 넓다          | ひろい       |
| 降る         | 降(내리다) + る          | 阝(언덕)              | 아래로 떨어짐           | 내리다(비/눈) | ふる         |
| 洗う         | 洗(씻다) + う            | 氵(물)                | 물로 씻어냄             | 씻다          | あらう       |
| 明るい       | 明(밝다) + るい          | 日(해)                | 해가 떠 밝아짐          | 밝다          | あかるい     |
| 押す         | 押(누르다) + す          | 扌(손)                | 손으로 누름             | 누르다        | おす         |
| 病院         | 病(병) + 院(원)          | 疒(병), 阝(언덕)      | 병을 치료하는 기관      | 병원          | びょういん   |
| 銀行         | 銀(은) + 行(행)          | 金(쇠), 行(가다)      | 은행업을 하는 곳        | 은행          | ぎんこう     |
